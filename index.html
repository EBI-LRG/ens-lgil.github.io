Advantages of using LRG sequences

  • Provide a unique stable sequence record with an ID and no versioning
  • Provide a coordinate system for permanent and unambiguous reporting of disease-causing variants
  • Independent of updates to the reference genome assembly
  • Important for bidirectional data exchange and integration
  • Compiled and maintained by the EBI and NCBI and so will be maintained in perpetuity
  • Integrated into the Ensembl and NCBI genome browsers to benefit from visualisation and existing annotation
  • Can be used as the reference sequence for HGVS nomenclature so that variant descriptions remain stable
  • Support legacy exon and amino acid numbering schemes
  • Supported by the Mutalyzer sequence variant nomenclature checker
